Note: How Git knows where to push and never confuses repos
~ When you run 'git init' the first time in any folder, for example in this one, it initialises a git repository in the folder -- i.e. it creates a hidden .git folder inside the project folder on computer. That folder contains the remote URL — the address of that specific repo on GitHub. When you run git push command from inside that folder for example, our quotation-tool folder here, Git reads that folder's remote URL and pushes only there. It has no awareness of your other projects at all."
